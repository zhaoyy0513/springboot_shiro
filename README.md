# springboot_shiro
Basic spingboot_shiro integration

* 此部分springboot版本为2.1.9Realease
* 数据库版本mysql8
* 默认登录可将项目中的数据库脚本导入后查看user表即可，登录名为:user_name字段 密码为user_pwd字段
* 集成mybatisPlus
* 集成404,500错误界面的友好跳转(后续用到可继续添加)
